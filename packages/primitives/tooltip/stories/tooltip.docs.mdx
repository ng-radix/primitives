# Tooltip

#### A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it.

## Features

- ✅ Global configuration (injection token) to control display delay globally.
- ✅ Opens when the trigger is focused or hovered.
- ✅ Closes when the trigger is activated or when pressing escape.
- ✅ Supports custom timings.

## Anatomy

```html
<ng-container rdxTooltipRoot>
    <button class="IconButton" rdxTooltipTrigger>+</button>

    <ng-template rdxTooltipContent [sideOffset]="8">
        <div class="TooltipContent" rdxTooltipContentAttributes>
            Add to library
            <div class="TooltipArrow" rdxTooltipArrow></div>
        </div>
    </ng-template>
</ng-container>
```

## Import

Get started with importing the directives:

```typescript
import {
  RdxTooltipRootDirective,
  RdxTooltipRootTrigger,
  RdxTooltipContentDirective,
  RdxTooltipContentAttributesDirective,
  RdxTooltipArrowDirective
} from '@radix-ng/primitives/tooltip';
```

or

```typescript
import { RdxTooltipModule } from '@radix-ng/primitives/tooltip';
```

## API Reference

## RdxTooltipConfig

<Markdown>
    {`
  | Prop | Type | Default | Description
  | ----- | ----- | ----- | -----
  | delayDuration | number | 700 | The duration from when the mouse enters a tooltip trigger until the tooltip opens.
  | skipDelayDuration | number | 300 | How much time a user has to enter another trigger without incurring a delay again.
  `}
</Markdown>

## RdxTooltipRootDirective

Contains all the parts of a tooltip.

<ArgTypes of={RdxTooltipRootDirective} />

## RdxTooltipTriggerDirective

The button that toggles the tooltip. By default, the TooltipContent will position itself against the trigger.

<Markdown>
    {`
   | Data attribute | Value
   | ----- | -----
   | [data-state] | <code>"closed" | "delayed-open" | "instant-open" (type RdxTooltipState)</code>
   `}
</Markdown>

## RdxTooltipContentDirective

The component that pops out when the tooltip is open.

<ArgTypes of={RdxTooltipContentDirective} />

<Markdown>
    {`
   | Data attribute | Value
   | ----- | -----
   | [data-state] | <code>"closed" | "delayed-open" | "instant-open" (enum RdxTooltipState)</code>
   | [data-side] | <code>"left" | "right" | "bottom" | "top" (enum RdxSide)</code>
   | [data-align] | <code>"start" | "end" | "center" (enum RdxAlign)</code>
   `}
</Markdown>

## RdxTooltipContentAttributesDirective

Directive state attributes

<Markdown>
    {`

   `}

## RdxTooltipArrowDirective

An optional arrow element to render alongside the tooltip. This can be used to help visually link the trigger with the TooltipContent. Must be rendered inside TooltipContent.
