# Toggle

#### A two-\_state button that can be either on or off.

<Canvas
  sourceState="hidden"
  of={ToggleDirectiveStories.Default}
/>

## Features

- ✅ Full keyboard navigation.
- ✅ Can be controlled or uncontrolled.

## Import

Get started with importing the directives:

```typescript
import { RdxToggleRootDirective } from '@radix-ng/primitives/toggle';
```

## Examples

```html
<button
  ToggleRoot
  aria-label="Toggle italic"
>
  <icon />
</button>
```

## API Reference

### RdxToggleRootDirective

<ArgTypes of={RdxToggleRootDirective} />

<Markdown>
  {`
  | Data Attribute | Value |
  | ----------- | --------- |
  | [data-state]       | "on" or "off"   |
  | [data-disabled]    | Present when disabled      |
  `}
</Markdown>

## Accessibility

### Keyboard Interactions

<Markdown>
  {`
  | Key | Description |
  | ----------- | --------- |
  | Space       | Activates/deactivates the toggle.       |
  | Enter       | Activates/deactivates the toggle.  |
  `}
</Markdown>
